 # 📖 English Reader — 英文文章阅读器
 
 一个纯前端的英文文章阅读器，帮助提升英文阅读水平。粘贴英文文章即可开始阅读，点击不会的单词查看释义，一键翻译整篇文章，收藏生词便于复习。
 
 ## 功能
 
 - **分词阅读** — 文章按单词拆分为可点击的词块，标点保留，段落结构完整
 - **点击查词** — 点击任意单词弹出浮动卡片，显示音标 + 中文翻译 + 英文定义 + 例句
 - **整篇翻译** — 一键开启后每句原文下方显示中文译文（需配置 API）
 - **生词收藏** — 点击单词卡片中的收藏按钮，将生词保存到生词本，带有原文上下文
 - **生词本** — 右侧滑入面板，支持搜索、删除、导出 TXT、清空
 - **设置面板** — API Key 配置、翻译目标语言、正文字号调节
 
 ## 使用说明
 
 1. **打开应用**
    - 直接双击 `index.html` 在浏览器中打开即可使用
    - 推荐使用 Chrome、Edge、Safari 等现代浏览器
 
 2. **粘贴文章**
    - 在输入区域粘贴任意英文文章（新闻、散文、故事等）
    - 点击 "📖 开始阅读" 或按 `Ctrl+Enter`（Mac: `Cmd+Enter`）
 
 3. **阅读查词**
    - 每个单词都是可点击的，鼠标悬停时有蓝色下划线提示
    - 点击单词 → 弹出浮动卡片，显示音标、释义和例句
    - 再点一次同一单词或点击页面空白处关闭卡片
    - 已收藏的单词右侧会显示 ★ 标记
 
4. **整篇翻译（可选）**
    - 点击顶部工具栏 "🌐 翻译" 按钮，每句下方会显示中文翻译（默认使用 MyMemory 免费翻译）
    - 可在设置中配置有道智云 API 获得更高质量的翻译
    - 再次点击可隐藏翻译

 5. **管理生词**（接前）
    - 查词卡片中点击 "💾 收藏单词" 将生词保存到生词本
    - 点击顶部工具栏 "📖 生词本" 打开侧栏查看所有收藏的单词
    - 支持搜索、单个删除、导出 TXT、清空全部
 
## 配置有道智云 API（可选）

 有道智云提供高质量的中文翻译服务。**不配置也能使用**，应用默认使用 MyMemory 免费翻译服务为你提供中文释义和整篇翻译。配置有道智云后可获得更准确的翻译结果。

### 申请步骤

 1. 访问 [有道智云 AI 开放平台](https://ai.youdao.com)
 2. 注册账号并登录
 3. 进入"控制台" → "我的应用" → "创建应用"
 4. 应用名称任意填写，选择服务: "文本翻译服务"
 5. 创建成功后，在应用详情页获取 **AppKey** 和 **AppSecret**
 6. 打开 English Reader → 点击 "⚙ 设置" → 填入 AppKey 和 AppSecret
 7. 点击 "🔄 测试连接" 验证配置是否正确
 8. 点击 "💾 保存设置"
 
 ### 费率说明
 
 - 有道智云新用户有免费额度（通常 50 元体验金）
 - 文本翻译按字符数计费，价格约为 50 元 / 100 万字符
 - 个人学习使用通常完全在免费额度内
 
## 技术说明

- **技术栈**: 纯 HTML + CSS + JavaScript（无第三方依赖）
- **数据存储**: 所有数据保存在浏览器 localStorage，关闭页面不会丢失
 - **词义查询**: Free Dictionary API（英文定义）+ MyMemory API（中文翻译，免费，无需 Key），可选升级为有道智云
 - **整篇翻译**: MyMemory API（默认，免费）+ 有道智云（可选升级）
- **翻译缓存**: 翻译结果缓存 24 小时，避免重复请求
- **单词缓存**: 每个单词 30 秒内不重复请求 API
 - **注意**: MyMemory 免费版每天约 100 次请求配额，个人阅读使用通常足够
